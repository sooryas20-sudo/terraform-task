# Terraform Task - Launch EC2 in Two Regions

## Task Overview
Launch Linux EC2 instances in two AWS regions using a single Terraform file.

## Tech Stack
- AWS EC2
- Terraform
- AWS CLI

## Regions Used
| Region | Location | Instance Name |
|--------|----------|---------------|
| ap-southeast-2 | Sydney, Australia | terraform-sydney |
| ap-south-1 | Mumbai, India | terraform-mumbai |

## What is Terraform?
Terraform is an Infrastructure as Code (IaC) tool. Instead of manually
clicking in AWS Console, you write code to create and manage infrastructure.
